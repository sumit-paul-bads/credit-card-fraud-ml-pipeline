# Credit Card Fraud Detection - ML Pipeline

## 📌 Overview
This project builds an end-to-end machine learning pipeline to detect fraudulent credit card transactions.

## ⚙️ Features
- Data preprocessing and EDA
- Handling class imbalance using SMOTE
- Stratified cross-validation
- Multiple model comparison
- Evaluation using ROC-AUC, Precision, Recall

## 🧠 Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting

## 🏆 Final Model
Random Forest was selected based on balanced precision and recall performance.

## 📊 Key Insights
- Dataset is highly imbalanced
- Accuracy is not a reliable metric
- Recall and ROC-AUC are critical for fraud detection

## 🚀 Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Imbalanced-learn
- Matplotlib, Seaborn
